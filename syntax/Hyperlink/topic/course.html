Heading

heading adalah bagian headline utama pada sebuah website

fungsi heading sendiri dapat membuat artikel menjadi lebih ternotice

heading juga dapat membuat artikel lebih mudah dibaca

heading juga dapat membuat artikel lebih mudah dipahami






Text Stylish

Text Stylish adalah teks yang memiliki gaya atau tampilan khusus

Text Stylish dapat digunakan untuk menekankan atau mempercantik teks

Contoh penggunaan Text Stylish adalah dengan menggunakan tag <strong> untuk teks tebal, <em> untuk teks miring, dan <u> untuk teks bergaris bawah.






Hyperlink

Hyperlink adalah tautan yang menghubungkan satu halaman web dengan halaman web lainnya

Hyperlink dapat digunakan untuk navigasi antar halaman web

Contoh penggunaan Hyperlink adalah dengan menggunakan tag <a href="URL"> untuk membuat tautan ke halaman web lain.






Ordered List

Ordered List adalah daftar yang memiliki urutan atau nomor

Ordered List dapat digunakan untuk menyusun informasi secara terstruktur

Contoh penggunaan Ordered List adalah dengan menggunakan tag <ol> untuk membuat daftar terurut.
