List Bahasa Pemrograman

  • HTML
    1. Hypertext MarkupLanguage
    2. Bahasa pemrograman dasar
    3. Sebagai bahan utama dalam pembuatan website
  • CSS
    1. Cascading Style Sheet
    2. Membuat website lebih berwarna
    3. diibaratkan seperti penambahan atribut untuk HTML dibagian design
  • Javascript
    1. Membuat website lebih hidup
    2. diibaratkan seperti kerangka tulang pada manusia
    3. dapat lebih banyak interaksi di websitenya
